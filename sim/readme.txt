// Files
1. src : source code
2. tbh : files for testbench
3. sim : files for compilation, simulation and waveform
   3.1 Makefile
       (a)The Makefile is used for compilation, simulation and waveform viewing.
       (b) The commands are
             make help   : Show help 
             make comp   : Compile code
             make sim    : Simulate code
             make wave   : Open waveform
	     make clean  : Clean files
   3.2 design.list
       (a)The file includes all files of source code.
   3.3 bench.list
       (a)The file includes all files of test bench.
